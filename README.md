# Codex Chess

An in-browser chess experience powered entirely by vanilla HTML, CSS, and JavaScript. No build tooling or backend is required—open `index.html` in any modern browser and start playing.

## Features
- Interactive chessboard with drag-free, click-to-move controls
- Legal move generation with automatic highlights for available destinations
- Check, checkmate, stalemate, and pawn promotion detection
- Move history log with readable descriptions
- Responsive layout with a dark, modern aesthetic

> **Note:** Castling and en passant are not implemented yet. All other core rules are supported.

## Development
1. Clone or download this repository.
2. Open `index.html` directly in a browser (double-click or use `open index.html` on macOS).
3. Edit `styles.css` or `app.js` to customize visuals or behavior—changes reload instantly on refresh.

No additional tooling or package managers are needed.
